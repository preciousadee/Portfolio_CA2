Project Deliverables

This page outlines the Bootstrap components and dynamic features included in my portfolio website, as required for the CA2 submission.

1. Bootstrap Components Used

  • Bootstrap Navbar: Used for the top navigation menu.
  • Bootstrap Grid System: Used for responsive layout throughout the website.
  • Cards: Used for Projects, Education and Experience sections.
  • Progress Bars: Used in the Skills section to visually show skill proficiency.
  • Buttons: Used for navigation, project filtering, and opening the contact modal.
  • Modal Window: Bootstrap modal used for Contact Information pop-up.
  • Responsive Images: Profile image and work logos using Bootstrap classes.

2. Dynamic Features Added (JavaScript)

  • Dark Mode Toggle: Allows the user to switch between light and dark themes.
  • Project Filter System: Filters the project cards by category using buttons.
  • Dynamic Greeting Message: Automatically updates greeting depending on time of day.
  • Smooth Section Scrolling: Enhances user experience when navigating the page.
  • Modal Trigger & Close Behaviour: Contact information opens in a pop-up modal.
  • Active Navigation Highlight: JS highlights the current section in the navbar.
